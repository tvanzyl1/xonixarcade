# 06 — Balancing and Finish Pass

## Final balancing goals
- movement feels responsive, not slippery
- line drawing is risky but fair
- enemy speed is readable
- wins are achievable without feeling trivial
- versus mode feels competitive and not chaotic nonsense

## Test checklist
- can the player always start a line from safe territory?
- does reconnecting always close correctly?
- are regions with enemies excluded from capture?
- does enemy contact with a live trail reliably punish the player?
- are HUD values always accurate?
- does pause work correctly?
- do both game modes fully restart without stale state?
- is the game fun in the first minute?

## Polish checklist
- no placeholder text in menus
- no tiny unreadable HUD text
- no muddy colour overlap between players
- no harsh abrupt screen transitions
- no silent confusing deaths
- no ambiguous winner screen

## Nice-to-have extras if time allows
- combo system for fast captures
- enemy variety
- level themes / colour variants
- replay-ready seed or difficulty option
- small attract-mode animation on menu

## Delivery expectation
The result should feel like a compact polished browser arcade game with two clear modes, not just a technical demonstration.
