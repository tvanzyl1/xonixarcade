# 03 — Solo vs Computer Mode

## Goal
The player must outplay computer enemies and capture a target amount of territory.

## Baseline rules
- 3 lives
- start with 1 enemy and ramp up
- target capture threshold: 75%
- each completed capture awards points
- clearing a round advances difficulty

## Difficulty scaling suggestions
Per round, increase one or more of:
- enemy count
- enemy speed
- target territory requirement
- hazard frequency

## Optional pickups
Only add if scope remains clean:
- slow enemies briefly
- shield for one hit
- trail speed boost
- bonus score token

## UX needs
Display clearly:
- lives
- score
- captured %
- round / level
- pause button

## Feedback moments
- major celebration when target % is reached
- warning pulse when an enemy is near active trail
- clear death burst and respawn moment
